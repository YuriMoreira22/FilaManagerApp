# FilaManagerApp
programa que consegue fazer gerenciamento de arquivos criado em java, esse codigo permite o usuario interagir com arquivos e pastas do computador por meio de um menu no terminal. Com ele é possivel: abrir arquivos ou diretorios renomear arquivos ou pastas mover arquivos de um lugar para o outro copiar arquivos excluir arquivos O programa funciona em loop, exibindo um menu até o usuario decidir sair. Ele utiliza entradas do teclado para receber os caminhos dos arquivos e executa as operações usando bibliotecas do java específicas para manipulação de arquivos. Além disso, o sistema registra logs de todas as ações realizadas e trata possiveis erros(como caminhos invalidos ou falta de permissão) informando o usuario que há algo errado.
